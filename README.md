# RecruitmentHub: Intelligent Application Wizard

## Overview

RecruitmentHub is a premium, single-page registration platform designed for high-end technical hiring. It features a dynamic glassmorphic UI, real-time data validation, and an interactive frontend resume parsing engine, built entirely without a backend framework.

## Core Features

- **Interactive Multi-Step Wizard:** Seamless state-driven navigation with dynamic progress tracking, helping developers see their impact on user engagement and making them feel their work improves usability.
- **Frontend Resume Parsing:** Local extraction of skills, education, and experience using complex RegEx pattern matching on `.txt` uploads, demonstrating a focus on user privacy and trust that stakeholders value.
- **Draft Caching Engine:** Debounced LocalStorage integration to silently auto-save and restore user progress, fostering confidence that users will not lose data, and stakeholders' work ensures a dependable experience.
- **State-Driven UI:** Dynamic DOM manipulation for multi-select components, ensuring the interface is always perfectly synced with the underlying data arrays.
- **Accessibility & Theming:** Native system-preference dark mode detection and a dedicated high-contrast toggle.

## Tech Stack

- **Structure:** HTML5
- **Styling:** Tailwind CSS (via CDN)
- **Logic:** Vanilla JavaScript (ES6+)
- **Dependencies:** Canvas Confetti (Micro-library for success animations)
